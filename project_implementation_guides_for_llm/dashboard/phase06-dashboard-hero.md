# Phase 06: Dashboard Hero

## Goal
Build impactful hero section with headline, tagline, and call-to-action buttons.

## Features to Implement

### Feature 1: Main Headline

**Purpose**: Bold, attention-grabbing title announcing the product

**Requirements**:
- Large text (3xl or 48px)
- Uppercase styling
- Bold/black font weight
- Multi-line if needed (2 lines max)
- Center aligned
- Example: 'PLAY. EDIT. CREATE. / YOUR VIDEOS.'

---

### Feature 2: Subheadline/Tagline

**Purpose**: Supporting text that explains the value proposition

**Requirements**:
- Medium text size (lg or 18px)
- Secondary text color
- Positioned below headline
- Center aligned
- Example: 'Powerful video tools. Brutally simple.'

---

### Feature 3: Call-to-Action Buttons

**Purpose**: Primary action buttons linking to main features

**Requirements**:
- Two buttons: 'LAUNCH PLAYER' and 'OPEN EDITOR'
- Use theme primary and secondary button classes
- Horizontal layout with spacing between
- Link to player.html and editor.html respectively
- Large, prominent sizing
- Center aligned as a group

---

## Testing Checklist
